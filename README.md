🚀 使用指南
1. 资料获取
直接在仓库中点击对应 PDF 文件，在线查看或下载到本地
建议按「课程→考纲→笔记 / 复习重点→工具 / 实战」的顺序使用，先明确考点，再针对性学习
2. 分阶段学习路线
表格
阶段	核心学习方向	重点资料
大一	数学基础 + 通识 + 编程入门	数分、复变、概率论、物理、Python、Linux 入门
大二	计算机核心基础 + 硬件入门	计算机网络、操作系统、数据结构、数据库、单片机、Keil 入门
大三	方向深化 + AI / 工程 + 工具进阶	算法设计、软件工程、图像处理、PyTorch、Git/Makefile、PCB 设计
备考	期末 / 考研 / 求职	对应课程考纲 + 复习重点 + 面试题（MySQL/Git 等）
3. 高效学习方法
先看考纲 / 大纲：明确学习 / 复习范围，避免盲目学习
再看笔记 / 教程：梳理知识点，搭建完整知识框架
结合实战 / 项目：用工具类资料（Git、Makefile、Keil 等）落地实践
最后刷题复盘：用复习重点、题型归纳巩固知识点，查漏补缺
